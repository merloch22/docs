---
title: 'Editado'
---

Este webhook é acionado quando uma mensagem é editada. Ele fornece informações detalhadas sobre a mensagem que foi modificada, incluindo dados sobre o remetente, destinatário, o conteúdo da mensagem editada e outros detalhes relevantes.

### Parâmetros

- **`tipo`**: Se o evento foi disparado pelo dispositivo configurado (ou seja, se o dispositivo é o remetente), o valor será `"Me"`. Se o evento estiver sendo recebido pelo dispositivo configurado (ou seja, se o dispositivo é o destinatário), o valor será `"Chat"`.
- **`nome`**: Nome do remetente da mensagem original antes da edição.
- **`destinatario`**: Número de telefone do destinatário da mensagem, ou seja, para quem a mensagem foi enviada antes de ser editada.
- **`remetente`**: Número de telefone do remetente da mensagem original, ou seja, de quem a mensagem foi enviada antes da edição.
- **`idMessagem`**: Identificador único da mensagem editada.
- **`data`**: A data em que a mensagem foi editada.
- **`horario`**: A hora exata em que a mensagem foi editada, no formato `HH:MM:SS`.
- **`tipoMensagem`**: Tipo da mensagem. Para mensagens editadas, o valor será `"editedMessage"`.
- **`mensagem`**: O novo conteúdo da mensagem após a edição.
- **`evento`**: Descrição do evento disparado. No caso de uma mensagem editada, será `"editada"`.
- **`idMessagemInteragida`**: O identificador único da mensagem que foi editada. Esse campo serve para identificar qual mensagem foi modificada.
