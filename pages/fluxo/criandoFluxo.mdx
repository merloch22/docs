---
title: "Criando um Fluxo"
---

Para criar um novo fluxo, certifique-se de que você possui um dispositivo ativo e a API devidamente cadastrada. Após concluir essas etapas, siga as instruções abaixo:

1. Clique em **"+ Criar Fluxo"**.

2. Defina um nome para o seu projeto de fluxo.

3. Clique em **"Criar agora"** para finalizar a configuração inicial.

![](/images/fluxo/image1.png)

Após a etapa acima o fluxo vai esta disponível para ser acessado:
