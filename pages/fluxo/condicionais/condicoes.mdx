As condições podem ser utilizadas de muitas formas, para criar uma aplicação mais robusta e eficaz, interagindo e com os resultados do usuário, vejamos um exemplo abaixo utilizando um input para determinar a condição:

Neste Exemplo abaixo criamos um input com duas alteranativas. O "\n" é utilizado para quebras de linha entao o resultado impresso será:

1 - Financeiro

2 - Compras

![](/images/fluxo/image9.png)

Após isso vamos criar nossa variavel para armazenar a resposta do usuario:

![](/images/fluxo/image10.png)

Ao criar a variável agora, vamos criar nossas condição que vai determinar o fluxo de nosso chat, por exemplo se a viaravel que criamos anteriormente 'resposta-menu' for igual a 1 vai para o financeiro.
