title: "Tipos de Inputs para Interação com o Usuário"
---

#### **Texto**

O "input" de texto é o mais comum e versátil, permitindo que o usuário insira respostas em formato textual. Perguntas gerais ou abertas são frequentemente utilizadas neste formato para coletar informações variadas.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/tipos-de-inputs-para-interacao-com-o-usuario#numero)**Número**

O "input" de número é utilizado para coletar respostas que devem ser expressas em formato numérico. Isso é útil para capturar valores quantitativos, como idade, quantidade de itens, entre outros.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/tipos-de-inputs-para-interacao-com-o-usuario#e-mail)**E-mail**

O "input" de e-mail é destinado a coletar endereços de e-mail válidos. É fundamental para registrar informações de contato e comunicação com o usuário de forma eficaz.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/tipos-de-inputs-para-interacao-com-o-usuario#data)**Data**

O "input" de data é utilizado para capturar datas no formato específico, garantindo a precisão das informações relacionadas a eventos, compromissos ou prazos.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/tipos-de-inputs-para-interacao-com-o-usuario#telefone)Telefone

O "input" de telefone é projetado para coletar números de telefone, seguindo um formato padrão para garantir a consistência e a possibilidade de contato posterior, se necessário.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/tipos-de-inputs-para-interacao-com-o-usuario#botao-experimental)Botão (Experimental)

Embora ainda esteja em fase experimental, o "input" de botão é convertido em texto com quebras de linhas.&#x20;
