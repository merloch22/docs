---
title: "Configurando Fluxo"
---

A configuração do fluxo é essencial para garantir uma experiência de qualidade para os usuários que interagem com ele.

1. Acesse a aba do fluxo e clique em **"Configurações"**.

2. Ajuste as opções conforme as necessidades do seu projeto.

Certifique-se de revisar todas as configurações para proporcionar uma interação eficiente e satisfatória para os usuários.

![](/images/fluxo/image3.png)

#### Tempo de Intervalo:

* **Definição:** Tempo estabelecido após o usuário finalizar um fluxo, durante o qual o mesmo usuário precisa esperar antes de iniciar um novo fluxo.

* **Objetivo:** Evitar iniciação imediata de um novo fluxo após a finalização, permitindo um intervalo para possíveis novas interações.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/configurando-fluxo#tempo-de-expiracao-do-fluxo)&#xA;Tempo de Expiração do Fluxo:

* **Definição:** Limite máximo de tempo para concluir um fluxo de conversa. Se o usuário não finalizar o fluxo nesse tempo, a conversa expira automaticamente.

* **Finalidade:** Evitar que uma conversa fique indefinidamente ativa caso o usuário não conclua o fluxo, permitindo o reinício ou abertura de uma nova conversa.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/configurando-fluxo#pausa-de-fluxo-no-servidor)&#xA;Pausa de Fluxo no Servidor:

* **Definição:** Mensagem ou comando que interrompe o autoatendimento do chatbot quando acionado, evitando a interferência na conversa.

* **Utilização:** Útil quando é necessário intervir manualmente, pausando o chatbot temporariamente para que ele não envie mensagens automáticas.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/configurando-fluxo#pausa-de-fluxo-pelo-cliente)&#xA;Pausa de Fluxo pelo Cliente:

* **Definição:** Mensagem específica ou comando enviado pelo cliente para interromper o fluxo do chatbot.

* **Aplicação:** Permite que o usuário interrompa o autoatendimento ao enviar uma mensagem particular, indicando o desejo de falar com um atendente humano.

#### [](https://digitaiss-organization.gitbook.io/documentacao-menuia-wordpress/chatbot-fluxo/configurando-fluxo#desativacao-para-todos-os-usuarios)&#xA;Desativação para Todos os Usuários:

* **Definição:** Configuração que desativa o chatbot, impedindo-o de responder a qualquer mensagem.

* **Propósito:** Útil em situações específicas em que é necessário desativar completamente o chatbot para todos os usuários.
