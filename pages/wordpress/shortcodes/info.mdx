---
title: "Shortcode Informações Pessoais"
---

**`[menuia_order_name]`**

* **Descrição:** Este shortcode exibe o campo onde os clientes podem inserir seu nome ao fazer um pedido. Ao utilizar este shortcode em seu site, será mostrado um campo para que os clientes digitem seus nomes durante o processo de pedido.

**`[menuia_order_phone]`**

* **Descrição:** Este shortcode gera um campo para os clientes inserirem seus números de telefone ao fazerem um pedido. É útil para coletar informações de contato dos clientes para facilitar a comunicação sobre os pedidos realizados.

**`[menuia_order_cellphone]`**

* **Descrição:** Similar ao anterior, este shortcode cria um campo destinado a receber os números de celular dos clientes. Isso pode ser útil para comunicações específicas ou para diferenciar entre números de telefone fixo e celular.

**`[menuia_order_email]`**

* **Descrição:** Ao usar este shortcode, um campo será exibido para que os clientes insiram seus endereços de e-mail durante o processo de pedido. Essa informação é vital para o envio de confirmações de pedidos e outras comunicações relacionadas.

**`[menuia_order_cpf]`**

* **Descrição:** Esse shortcode cria um campo específico para que os clientes insiram seus números de CPF (Cadastro de Pessoas Físicas). Este campo é opcional por padrão no woocommerce.

**`[menuia_order_cnpj]`**

* **Descrição:** Similar ao anterior, mas destinado a empresas ou pessoas jurídicas, este shortcode oferece um campo para inserção de CNPJ (Cadastro Nacional da Pessoa Jurídica). Este campo é opcional por padrão no woocommerce.
