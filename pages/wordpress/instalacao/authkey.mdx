---
title: "Autenticando Authkey"
---

Após obter a sua Authkey, siga estes passos para integrá-la ao seu site WordPress usando o Plugin Menuia:

**Navegue até a Configuração do Plugin Menuia**

No menu lateral, clique em "Menuia" para acessar as configurações do plugin.

Selecione a opção "Infos" para inserir suas informações de autenticação.

![](/images/wordpress/image5.png)

**Insira Sua Authkey**

Na seção de informações, localize o campo designado para a Authkey.

**Passo 4: Preencha com Sua Authkey**

Digite ou cole sua Authkey no campo apropriado.

![](/images/wordpress/image6.png)

**Passo 5: Autenticação**

Clique no botão "Autenticar" para validar sua Authkey. Isso estabelecerá a conexão segura entre seu site WordPress e a API da Menuia.
