## Descrição

Este endpoint permite obter a imagem de um usuário ou grupo no sistema do chatbot. A operação é ativada pelo parâmetro `obterImagem`.

### Parâmetros

\* `appkey`: Chave da aplicação.  
\* `authkey`: Chave de autenticação do usuário.  
\* `to`: Número do usuário (ex: `5581989769960`) ou ID do grupo (ex: `120363180117806010@g.us`).  
\* `obterImagem`: Deve ser `"true"` para solicitar a imagem.  
\* `message`: Deve ser `"false"` para realizar a operação.  

### Exemplo de requisição:

```json
{
  "appkey": "SUA_APPKEY_AQUI",
  "authkey": "SUA_AUTHKEY_AQUI",
  "to": "5581989769960 ou 120363180117806010@g.us",
  "obterImagem": "true",
  "message": "false"
}
