## Descrição

Este endpoint permite criar um novo agente de IA associado à sua conta, informando o nome do agente e o dispositivo que o hospeda.

### Parâmetros

* `authkey`: Chave de autenticação da conta.
* `device`: Dispositivo que irá hospedar o agente (ex: "servidor").
* `agent_name`: Nome do agente que será criado.

### Exemplo de Requisição:

```json
{
  "authkey": "A29gzpIJyLWvxg2opJaq70AscJ08PWGLfydphRmjAJtY9IQLoz",
  "device": "servidor",
  "agent_name": "AGENTE 2"
}
