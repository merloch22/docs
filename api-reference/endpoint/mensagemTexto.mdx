## Descrição

Este endpoint permite enviar uma mensagem para um número ou grupo de forma simples.

### Parâmetros

* `appkey`: Chave de aplicação.

* `authkey`: Chave de autenticação do usuário.

* `to`: Número de telefone ou ID do grupo.

* `message`: Conteúdo da mensagem.

### Exemplo de requisição:

```json
{
  "appkey": "SUA_APPKEY_AQUI",
  "authkey": "SUA_AUTHKEY_AQUI",
  "to": "+5511999999999",
  "message": "Olá, esta é uma mensagem!"
}
```
