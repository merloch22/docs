---

## Descrição

Este endpoint permite atualizar as informações de um agente de IA existente, incluindo nome, dispositivo, webhook, voz, dias de trabalho, sistema e outras configurações.

### Parâmetros

* `id`: ID do agente de IA que será atualizado (informado na URL do endpoint).
* `authkey`: Chave de autenticação da conta.
* `cpf`: Booleano indicando se o CPF deve ser usado.
* `email`: Booleano indicando se o e-mail deve ser usado.
* `webhook`: URL do webhook que o agente irá usar.
* `device_id`: ID do dispositivo onde o agente está hospedado.
* `send_audio`: Booleano para enviar áudio.
* `receiver_audio`: Booleano para receber áudio.
* `voice`: Voz do agente ("m" para masculino, "f" para feminino).
* `agent_name`: Nome do agente.
* `system`: Descrição ou contexto do agente.
* `stop_bot`: Booleano indicando se o bot deve parar.
* `work_days`: Lista de dias da semana que o agente trabalha.
* `stop_client`: Booleano para parar atendimento ao cliente.
* `stop_server`: Booleano para parar atendimento no servidor.
* `time_pause`: Tempo de pausa entre interações.
* `blacklist`: Lista de palavras proibidas.
* `temperature`: Valor de temperatura para respostas do agente.

### Exemplo de `curl`:

```bash
curl --location --request PUT 'https://chatbot.menuia.com/api/v2/ia/35' \
--header 'Content-Type: application/json' \
--data '{
    "authkey": "SUA_AUTHKEY",
    "cpf": 1,
    "email": 0,
    "webhook": "https://meusite.com/webhook",
    "device_id": 4586,
    "send_audio": 1,
    "receiver_audio": 1,
    "voice": "m",
    "agent_name": "Lab IA",
    "system": "Você é uma Biomédica responsável por Laboratório de Análises Clínicas",
    "stop_bot": 0,
    "work_days": ["seg","ter","qua","qui","sex","sab","dom"],
    "stop_client": 0,
    "stop_server": 0,
    "time_pause": "00:30:00",
    "blacklist": ["palavra1","palavra2"],
    "temperature": 0.3
}'
