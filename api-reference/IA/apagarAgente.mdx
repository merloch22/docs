---
title: 'Apagar Agente de IA'
openapi: 'DELETE /api/v2/ia/{id}'
---

## Descrição

Este endpoint permite apagar um agente de IA existente, utilizando o `id` do agente e a chave de autenticação.

### Parâmetros

* `id`: ID do agente de IA que será apagado (informado na URL do endpoint).
* `authkey`: Chave de autenticação da conta.

### Exemplo de `curl`:

```bash
curl --location --request DELETE 'https://chatbot.menuia.com/api/v2/ia/34' \
--header 'Content-Type: application/json' \
--data '{
    "authkey":"A29gzpIJyLWvxg2opJaq70AscJ08PWGLfydphRmjAJtY9IQLoz"
}'
