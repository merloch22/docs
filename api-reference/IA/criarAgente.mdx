# Criar Agente de IA

### Headers

| Header | Tipo | Obrigatório |
|--------|------|------------|
| Content-Type | string | Sim |
| authkey | string | Sim |

### Parâmetros do Corpo (JSON)

| Parâmetro | Tipo | Obrigatório |
|-----------|------|------------|
| authkey | string | Sim |
| device | string | Sim |
| agent_name | string | Sim |

### Exemplo de Requisição

```bash
curl --location 'https://chatbot.menuia.com/api/v2/ia' \
--header 'Content-Type: application/json' \
--data '{
    "authkey":"SUA AUTHKEY",
    "device":"SEU DEVICE",
    "agent_name":"AGENTE 2"
}'
